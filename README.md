# Sistema de Gerenciamento de Estoque

Este projeto é um sistema simples em linguagem C para gerenciar o estoque de uma loja. Ele permite o controle de entrada e saída de produtos, assim como a atualização e listagem de itens.

## Funcionalidades

- **Adicionar Produto**: Adiciona novos produtos ao estoque, com ID, nome, preço e quantidade.
- **Editar Produto**: Edita as informações de um produto existente, identificando-o pelo ID.
- **Listar Produtos**: Exibe a lista completa de produtos cadastrados.
- **Persistência de Dados**: O sistema salva automaticamente os dados no arquivo `estoque.txt`.
